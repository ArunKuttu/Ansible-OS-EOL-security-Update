# Ansible-OS-EOL-security-Update
This ansible play book will check on the OS EOL in the office machines and send a mail alert to sysadmin team if their OS eol is expired

In this We have created 2 playbooks 

1) OS EOL checking and Alerting
2) Server/system security updates checking and Alerting

Play book names
1) check_os_eol.yml
2) check_security_updates.yml

   We have used ansible vault for authentication purpose
